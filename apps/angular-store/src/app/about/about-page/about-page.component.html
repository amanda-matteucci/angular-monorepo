Accordian Component:

Accordian components are great for displaying different pieces of information such as FAQs. Accordions are commonly used to reduce the need to scroll when presenting multiple sections of content on a single page.

Absolutely! There are three modifiers for the Accordian.

  • Borderless (not currently displayed)
  • Bordered
  • Multiselectable

By default the Accordian closes all other sections when a new one is clicked. When using the Multiselectable modifier, all previously clicked sections will stay open.

The Borderless modifier removes the thin line that is displayed around the sections of the Accordian while they are open.
